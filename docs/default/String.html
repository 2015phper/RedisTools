setSubsting (line 156)

sets a part of the string stored at key.

Substring starts at $startIndex (default to first position) and ends at the length of $string. Empty keys are set to the substring. Every offset given in start string and not already filled with characters is filled with null values.

ATTENTION: Keep in mind that string operations are byte base, so operations on utf-8 string will not deliver the expected characters.

  • return: - the new length of the string
