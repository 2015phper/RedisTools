push (line 53)

Adds a value to this ArrayList.

Per default the value is inserted at the end of the list (position at length). If $toHead is set to true the value is inserted as the first element of the List. Per default if the list does not already exist it is created on push. Setting $createNonExisting to false prevents this behaviour and only pushes elements to the list if it already exists.

  • return: - the new length of list or, false on error
  • access: public

  • string $value
  • boolean $toHead: - wether to push to the top of the list
  • boolean $createNonExisting: - wether to create the list if not already existing
